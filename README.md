## 📌 Project Description

Movie Tracker (CineTrack) is a web application that allows users to manage and track movies they have watched or plan to watch.

The system provides functionality for:

* User authentication (registration and login)
* Adding and managing movies
* Writing reviews and giving ratings
* Creating a personal watchlist

This project is developed as part of the Web Development course using Angular for the frontend and Django REST Framework for the backend.

---

##  Project Goals

The main goal of this project is to build a full-stack web application that demonstrates:

* Frontend-backend interaction via REST API
* User authentication with JWT
* CRUD operations for managing data
* Clean UI with basic user experience
